
The file channel_velocity.m.txt is a text file containing MATLAB code implementing the complete solution for an infinite solution of earthquakes. Input arguments and outputs are explained, and an example is provided, at the beginning of the code.

Reference: DeVries. P.R. and B.J. Meade, Earthquake cycle deformation in the Tibetan plateau with a weak mid-crustal layer, J. Geophys. Res., 118, 3101-3111.
